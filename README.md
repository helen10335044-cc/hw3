##Booking System 2 – Java 訂房系統

使用 Java 8 + Maven + MVC + DAO Pattern + MySQL 開發的訂房管理系統。<br>

本專案模擬真實訂房網站，包含：<br>
會員前台訂房<br>
多圖輪播房型預覽<br>
購物車結帳<br>
房間庫存控制<br>
防止重複訂房<br>
後台管理（產品 / 訂單）<br>
交易控制（Transaction）<br>
分頁查詢<br>
JasperReport PDF 報表<br>

##🚀 功能介紹
#👤 會員前台
會員註冊 / 登入
日期區間查詢
顯示該日期剩餘庫存
防止超賣
多圖輪播預覽
加入購物車
結帳下單
訂單查詢

#🛒 購物車功能
可選擇數量（1~2 間）
即時計算總價
下單前再次檢查庫存
Transaction 控制避免超賣

#🖼 多圖輪播
支援左右切換
自動播放（3秒切換）
ESC 關閉
左右方向鍵控制

#🛠 後台管理
管理員登入後可：
管理產品（CRUD）
管理訂單
列印產品報表
列印訂單報表（PDF）
分頁查詢
