# Booking System 2 – Java 訂房系統

使用 Java 8 + Maven + MVC + DAO Pattern + MySQL 開發的訂房管理系統。<br>

本專案模擬真實訂房網站，包含：<br>
會員前台訂房<br>
多圖輪播房型預覽<br>
購物車結帳<br>
房間庫存控制<br>
防止重複訂房<br>
後台管理（產品 / 訂單）<br>
交易控制（Transaction）<br>
分頁查詢<br>
JasperReport PDF 報表<br>
